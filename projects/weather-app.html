overview

The Weather App is a cross-platform application built using React Native. It provides users with real-time weather updates and forecast. The app is designed to be user-friendly and visually appealing. The clean UI makes the app easy for users to access the information they need. What started off as a school homework ended up into a neat app!

features

  • Real-time weather data from OpenWeatherMap API
  • Location-based weather updates
  • Forecast for the next 6 days
  • Works on Android, iOS, and Web
